## Objetivo del proyecto

Formalizar e implementar un subconjunto del lenguaje LISP llamado **MINILISP**, integrando teoría y práctica mediante:

- Definición léxica y sintáctica ( EBNF )
- Eliminación de azúcar sintáctica
- Semántica operacional estructural ( small-step )
- Implementación funcional en Haskell
